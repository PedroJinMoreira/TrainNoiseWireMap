# TrainNoiseWireMap
The program can be executed by up to 3 parameters:  
	bgcolor, wirecolor, pointcolor

```shellscript
./TrainNoiseWireTerrain
./TrainNoiseWireTerrain 0 0 0 0		
./TrainNoiseWireTerrain 1 1 1 1 0.25 0.25 0.25 1 0 0 0 1	
```
